# Products Client Application

### How to run with docker
- clone repo
- run `docker-compose up --build`
  
### How to run with locally
- clone repo
- run `npm install`
- define `.env` file and set the value `REACT_APP_BACKEND_URL='http://localhost:8080'`
- run `yarn start`
